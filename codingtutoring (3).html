Coding The Future
Dalton coding classes
Classes
Coding the Future currently offers two coding courses. These are one semester courses led by Dalton high schoolers for middle school students. Students who are interested in taking on a passion project will also have the oppurtunity to do so alongside a high school mentor.
1. Web Development: HTML, CSS, JS
In this course, students will learn how to code using the basic languages for web development. They will first learn how to use HTML elements and CSS styling to create a webpage. They will then implement javascript functions to make their websites come to life with interactive components, animations, and scrolling effects.
2. Art and Games with Python
In this course, students will learn how to make art and games using Python. They will start by experimenting with simple Python functions, and later move to using algorithms to make more intricate visuals.
The Mission
We are a club at Dalton where talented and motivated high school students came together to design two engaging classes aimed at introducing young students to computer programming.
The Problem
As middle school students we found ourselves using block code and other "kidified" versions of coding, but we were really more interested in something that felt more real -- coding languages that are actually used in the real world, and frameworks that are more versatile.
Our Solution
We are providing students with the oppurtunity we wish we had, allowing them to engage with coursework that is designed to be managable, but not dumbed down. Given the right instruction we know that middle school students are capable of understanding computer programming, and doing great things with it.
